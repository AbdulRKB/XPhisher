### Installation

- Just, Clone this repository -
```
$ git clone git://github.com/CyberTitus/XPhisher.git
```

- Change to cloned directory and run `main.sh` -
```
$ cd XPhisher
$ bash main.sh
```

### Dependencies

**`XPhisher`** requires following programs to run properly - 
- `php`
- `wget`
- `curl`
- `git`
